# فيتيوب
قراءة وإدارة دردشة youtube و twitch بطريقة يسهل الوصول إليها في عمليات البث الخاصة بك أو تلك الخاصة بالمبدعين المفضلين لديك.
[عرض عمل المنتج] (https://youtu.be/4XawJoBymPs)
## صفات
- الوضع التلقائي: قراءة رسائل الدردشة في الوقت الحقيقي باستخدام sapy5 voice
- واجهة غير مرئية: إدارة الدردشات من أي نافذة باستخدام أوامر لوحة مفاتيح بسيطة. يجب أن يكون لديك قارئ شاشة نشط.
- دعم القراء:
-NVDA
-الفجوس
-عيون النافذة
- سوبرنوفا
- الوصول إلى النظام
المتكلم -PC
-ZDSR
- إمكانية التهيئة حسب احتياجات المستخدم.
- ينشط او يعطل اصوات البرنامج.
- ينشط أو يلغي تنشيط الوضع التلقائي.
- تكوين قائمة الرسائل في الواجهة غير المرئية.
- تكوين تفضيلات sapy voice.
- تخصيص اختصارات لوحة المفاتيح العالمية
- قم بتغيير وضع قراءة الدردشات بسهولة: حدد ما إذا كنت تريد قراءة جميع الدردشات أو تلك الخاصة بفئة معينة فقط.
- حفظ حياتك في قسم المفضلة. كرر الدردشة عدة مرات كما تريد دون الحاجة إلى البحث عن الرابط مرة أخرى.
- أرشفة رسالة: مفيدة في وجود تذكيرات.
- يحفظ تاريخ الرسائل.
- ترجمة الدردشة المتدفقة إلى اللغة التي تريدها.

## اختصارات لوحة المفاتيح.
### استخدام واجهة غير مرئية.
| العمل | تركيبة المفاتيح |
| ------------------------- | ----------- |
| إسكات الصوت العاقل | السيطرة + p |
| المخزن المؤقت السابق | مفتاح بديل للسهم الأيسر |
| المخزن المؤقت التالي | مفتاح بديل للسهم الأيمن |
| العنصر السابق | التحول لأعلى سهم بديل |
| العنصر التالي | تحول بديل السهم لأسفل |
| العنصر الأولي | التحول إلى المنزل |
| عنصر النهاية | نهاية التحول البديل |
| أرشفة رسالة | التحول إلى |
| انسخ الرسالة الحالية | التحول البديل ج |
| يمسح مخزن مؤقت تم إنشاؤه مسبقًا | تحول بديل د |
| إضافة رسالة إلى المفضلة المخزن المؤقت | الشفتف |
| قم بتشغيل الوضع التلقائي أو إيقاف تشغيله | التحول البديل |
| إيقاف تشغيل أصوات البرنامج | بديل التحول ص |
| البحث عن كلمة بين الرسائل | تبديل s |
| يعرض الرسالة الحالية في مربع نص | تحول بديل v |

### في سجل الدردشة:
| العمل | تركيبة المفاتيح |
| ------------------------- | ----------- |
| تشغيل الرسالة المختارة | الفضاء |

### في قسم المفضلة:
| العمل | تركيبة المفاتيح |
| ------------------------- | ----------- |
| الوصول إلى ارتباط محدد | الفضاء |
