#vetubi
Leggi e gestisci le chat di YouTube e Twitch in modo accessibile nelle tue trasmissioni o in quelle dei tuoi creatori preferiti.
[demo funzionante del prodotto](https://youtu.be/4XawJoBymPs)
## Caratteristiche
- Modalità automatica: leggi i messaggi di chat in tempo reale utilizzando la voce sapy5
- Interfaccia invisibile: gestisci le chat da qualsiasi finestra utilizzando semplici comandi da tastiera. È necessario disporre di un lettore di schermo attivo.
- Lettori supportati:
-NVDA
- MASCELLE
-Finestra-Occhi
- Supernova
-Accesso al sistema
-PC Talker
-ZDSR
- Possibilità di configurazione secondo le esigenze dell'utente.
- attiva o disattiva i suoni del programma.
- attiva o disattiva la modalità automatica.
- configura l'elenco dei messaggi nell'interfaccia invisibile.
- Configura le preferenze della voce sapy.
- personalizzare le scorciatoie da tastiera globali
- cambia facilmente la modalità di lettura delle chat: decidi se vuoi leggere tutte le chat o solo quelle di una specifica categoria.
- salva la tua vita in una sezione dei preferiti. ripeti la chat tutte le volte che vuoi senza dover cercare nuovamente il link.
- archiviare un messaggio: utile per avere promemoria.
- salva una cronologia dei messaggi.
- tradurre la chat di uno streaming nella lingua che preferisci.

## Tasti rapidi.
### Utilizzo dell'interfaccia invisibile.
| azione | combinazione di tasti |
| ------------------------- | ----------- |
| Fai tacere la voce sapide | ctrl+p |
| tampone precedente | alt spostamento freccia sinistra |
| tampone successivo | alt spostamento freccia destra |
| Voce precedente | alt spostamento freccia su |
| Articolo successivo | alt spostamento freccia giù |
| Elemento iniziale | alt sposta casa |
| elemento finale | alt fine turno |
| archiviare un messaggio | altshift a |
| Copia il messaggio corrente | alt maiusc c |
| Cancella un buffer precedentemente creato | alt maiusc d |
| aggiungi messaggio al buffer dei preferiti | altshiftf |
| Attiva o disattiva la modalità automatica | alt shiftm |
| disattivare i suoni del programma | alt-shift p |
| cerca una parola tra i messaggi | altshift s |
| visualizza il messaggio corrente in una casella di testo | alt maiusc v |

### Nella cronologia chat:
| azione | combinazione di tasti |
| ------------------------- | ----------- |
| Riproduci messaggio selezionato | spazio |

### Nella sezione dei preferiti:
| azione | combinazione di tasti |
| ------------------------- | ----------- |
| accedere a un collegamento selezionato | spazio |
