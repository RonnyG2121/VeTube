#vetube
Read and manage youtube and twitch chat in an accessible way in your own broadcasts or those of your favorite creators.
[product working demo](https://youtu.be/4XawJoBymPs)
## Characteristics
- Automatic mode: Read chat messages in real time using sapy5 voice
- Invisible interface: Manage chats from any window using simple keyboard commands. You need to have an active screen reader.
- Readers supported:
-NVDA
-JAWS
-Window-Eyes
- Supernova
-System Access
-PC Talker
-ZDSR
- Possibility of configuring according to the user's needs.
- activates or deactivates the sounds of the program.
- activates or deactivates the automatic mode.
- configures the list of messages in the invisible interface.
- Configure the preferences of the sapy voice.
- customize global keyboard shortcuts
- change the mode of reading chats easily: decide if you want to read all the chats or only those of a specific category.
- save your live in a favorites section. repeat the chat as many times as you want without having to go looking for the link again.
- archive a message: useful for having reminders.
- saves a history of messages.
- translate the chat of a streaming to the language you like.

## Keyboard shortcuts.
### Using invisible interface.
| action | key combination |
| ------------------------- | ----------- |
| Silence the sapy voice | ctrl+p |
| previous buffer | alt shift left arrow |
| next buffer | alt shift right arrow |
| Previous item | alt shift up arrow |
| Next item | alt shift down arrow |
| Initial element | alt shift home |
| end element | alt shift end |
| archive a message | altshift to |
| Copy the current message | alt shift c |
| Clears a previously created buffer | alt shift d |
| add message to favorites buffer | altshiftf |
| Turn automatic mode on or off | alt shiftm |
| turn off program sounds | alt-shift p |
| search for a word among messages | altshift s |
| displays the current message in a text box | alt shift v |

### In chat history:
| action | key combination |
| ------------------------- | ----------- |
| Play selected message | space |

### In the favorites section:
| action | key combination |
| ------------------------- | ----------- |
| access a selected link | space |
