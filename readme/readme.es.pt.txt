#vetube
Leia e gerencie o chat do youtube e twitch de forma acessível em suas próprias transmissões ou nas de seus criadores favoritos.
[demonstração de funcionamento do produto](https://youtu.be/4XawJoBymPs)
## Caracteristicas
- Modo automático: leia as mensagens de bate-papo em tempo real usando a voz sapy5
- Interface invisível: gerencie chats de qualquer janela usando comandos de teclado simples. Você precisa ter um leitor de tela ativo.
- Leitores suportados:
-NVDA
-MANDÍBULAS
-Janela-Olhos
- Super Nova
-Acesso ao sistema
-PC Talker
-ZDSR
- Possibilidade de configuração de acordo com as necessidades do usuário.
- ativa ou desativa os sons do programa.
- ativa ou desativa o modo automático.
- configura a lista de mensagens na interface invisível.
- Configure as preferências da voz sapy.
- personalizar atalhos de teclado globais
- altere facilmente o modo de leitura dos chats: decida se deseja ler todos os chats ou apenas os de uma categoria específica.
- salve sua live em uma seção de favoritos. repita o chat quantas vezes quiser sem precisar procurar o link novamente.
- arquivar uma mensagem: útil para ter lembretes.
- salva um histórico de mensagens.
- traduza o chat de um streaming para o idioma que você gosta.

## Atalhos do teclado.
### Usando interface invisível.
| ação | combinação de teclas |
| ------------------------- | ----------- |
| Silencie a voz sapy | ctrl+p |
| buffer anterior | Alt shift seta para a esquerda |
| próximo buffer | Alt shift seta para a direita |
| Item anterior | Alt shift seta para cima |
| Próximo item | Alt shift seta para baixo |
| Elemento inicial | Alt shift home |
| elemento final | fim do turno alternativo |
| arquivar uma mensagem | Altshift para |
| Copie a mensagem atual | Alt shift c |
| Limpa um buffer criado anteriormente | Alt shift d |
| adicionar mensagem aos favoritos buffer | altshiftf |
| Ativar ou desativar o modo automático | alt shiftm |
| desligar os sons do programa | alt-shift p |
| procurar uma palavra entre as mensagens | altshift |
| exibe a mensagem atual em uma caixa de texto | Alt shift v |

### No histórico do chat:
| ação | combinação de teclas |
| ------------------------- | ----------- |
| Reproduzir mensagem selecionada | espaço |

### Na seção de favoritos:
| ação | combinação de teclas |
| ------------------------- | ----------- |
| acessar um link selecionado | espaço |
